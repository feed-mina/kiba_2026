### 주요 수치 (2026년 기준)

- 유량 조사지점 수: **144개소**
- 유량 지점당 단가: **84백만원**
- 2026 유량 사업 예산: **약 20.4억원**
- 보유 장비 총수: **408대**
- 권역별 컨테이너 임차 연간: 낙동강 25,830천원 / 금강 17,292천원 / 영산강 15,640천원 / 한강 6,138천원

## 2. 분석 플랜

### Phase A. 예산·단가 트렌드 분석

**목적**: 연도별(2021→2026) 예산 증감 패턴과 지점당 단가 안정성 파악

- [ ] 항목별 연도별 실행 예산 추출 (유량·유사량·토양수분·증발산·자동유량)
- [ ] 지점당 단가 변화 확인 (유량 84M원/지점 고정, 자동유량 운영 49→51M원 변화)
- [ ] 기관운영 예산 vs 사업 예산 비중 비교
- [ ] 잔액 예산 반영 전·후 차이 분석

### Phase B. 지점 1건당 원가 구조 분해

**목적**: 수문(유량) 조사 지점 1건의 투입 원가를 항목별로 분해

| 원가 항목 | 산출 방법 | 데이터 소스 |
|-----------|-----------|-------------|
| 직접인건비 | 단가 84M원 중 인건비 비율 추정 (표준품셈 참조) | 02_단가표, 03_지침 |
| 장비비 | ADCP·보트 단가 × 감가상각률 ÷ 연간 측정 횟수 | 05_장비 관련 |
| 교통비 (왕복) | 해당 지점 편도 교통비 × 2 × 투입 인원 | 04_조사거점 교통비 |
| 차량 운영비 | 월 임차료 / (지점 수 × 월 방문 횟수) | 06_업무차량 |
| 창고·시설 임대비 | 권역별 연간 컨테이너 임차료 ÷ 권역 내 지점 수 | 06_임대 관련 |
| 간접비 | 직접비 합계 × 관리비율(추정 15~18%) | 표준품셈 |

- [ ] 표준품셈 기준 인건비 산출 방식 확인 (03_지침 PDF)
- [ ] ADCP 장비 1대당 연간 감가상각비 계산
- [ ] 지역별(4개 권역) 교통비 평균 계산
- [ ] 지점 1건 기준 원가 추정 시트 작성

### Phase C. 지역별(권역별) 비교 분석

**목적**: 한강·낙동강·금강·영산강 4개 권역의 지점 운영 단가 차이 파악

- [ ] 권역별 컨테이너 임차비 / 지점 수 → 지점당 임대 원가
- [ ] 교통비 파일에서 권역별 평균 편도 교통비 추출
  - 낙동강(경상도): 안동·상주·영주 등 21,800~28,000원 수준
  - 금강(충청도): 단양·제천·충주 등 15,000~18,300원 수준
  - 영산강(전라도): 데이터 추가 확인 필요
  - 한강(수도권·강원): 1,500~23,000원 편차 크게 분포
- [ ] 권역별 원가 비교표 작성 (1건당 기준)

### 3-2. 핵심 UI 컴포넌트

| 컴포넌트 | 구현 방법 | 데이터 |
|----------|-----------|--------|
| KPI 카드 4개 | CSS grid, 애니메이션 카운터 | 총예산·지점수·단가·교통비 |
| 연도별 예산 스택 막대 | SVG 직접 렌더링 (Chart.js 불필요) | 01_사업별 예산 |
| 지점 원가 도넛 차트 | SVG stroke-dasharray | 원가 항목별 비율 |
| 지점 선택 드롭다운 | HTML select + JS 필터 | 04_조사지점 현황 |
| 교통비 수직 막대 | SVG | 04_조사거점 교통비 |
| 권역 비교 테이블 | HTML table + 색상 강조 | 운영비 통합 |
| 원가 계산기 | 입력폼 + JS 실시간 계산 | 단가표 |

### 3-3. 사용자 친화성 포인트

1. **직관적 탭 네비게이션**: 개요 → 지점별 드릴다운 → 권역 비교 순서
2. **지점 직접 검색**: 지점명 입력 시 교통비·권역 자동 표시
3. **원가 항목 호버 툴팁**: 각 비용 항목에 마우스 올리면 산출 근거 팝업
4. **권역별 색상 구분**: 한강(파랑)·낙동강(초록)·금강(노랑)·영산강(보라) 일관 적용
5. **인쇄/엑셀 내보내기**: 현재 뷰 기준 산출서 PDF·xlsx 생성 버튼
6. **모바일 반응형**: 768px 이하에서 카드 1열, 탭 스크롤
