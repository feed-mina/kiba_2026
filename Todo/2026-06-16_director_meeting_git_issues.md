# 원장님 요구사항 정리 (2026-06-16 미팅)

> 목적: 2026년 6월 16일 원장님 미팅에서 나온 주요 요구사항을 Git Issue 형식으로 정리합니다.
> 관리 방식: 각 항목은 GitHub Issue로도 등록하여 진행 상태를 추적합니다.

## 1. [인력 관리] 엔지니어링 협회 등록을 위한 자격증 및 요건 검토

**상세 내용:** 한국 엔지니어링 협회 및 건설 엔지니어링업 신규 등록을 위해 현재 보유 인력(40명)의 자격증, 학력, 경력을 대조하여 요건을 충족하는지 확인이 필요합니다.

**체크리스트:**
- [ ] 사업 관리 분야 등록을 위한 필수 인력 10명 요건 충족 여부 확인.
- [ ] 인력별 자격증/학위 기반 등급(초급/중급/고급/특급) 판정 및 매칭.
- [ ] 신규 등록 및 인력 추가를 위한 최종 인력 리스트 도출.

---

## 2. [자동화] 엑셀 현황판 - 구글 스프레드시트 자동 동기화

**상세 내용:** 각 팀에서 개별적으로 작업 중인 엑셀 시트(스마트 공장, 산학연 등)의 데이터를 구글 스프레드시트 현황판으로 매일 자동 업데이트하는 체계가 필요합니다.

**체크리스트:**
- [ ] 로컬 엑셀 파일 수정 시 구글 시트에 즉시 반영 또는 덮어쓰기 연동 방식 검토.
- [ ] 클로드(Claude)를 활용하여 엑셀 데이터를 분석하고 변수/함수 값으로 추출하여 자동화.
- [ ] 스마트 공장 사업 관리 시스템 등의 외부 사이트 현황 데이터를 자동으로 끌어오기.

---

## 3. [대가 산정] SW 대가 산정 가이드 자동 업데이트 및 데이터화

**상세 내용:** 소프트웨어 산업진흥협회에서 매년 공개하는 가이드라인(PDF)을 시스템에 연동하고, 최신 인건비 기준을 자동으로 반영해야 합니다.

**체크리스트:**
- [ ] 협회 홈페이지를 모니터링하여 최신 가이드라인 PDF 및 인건비 기준(8월 공개) 자동 수집.
- [ ] PDF 내 복잡한 기능 점수(FP) 산정 식을 분석하여 시스템 계산 로직에 자동 반영.
- [ ] 과거 버전(2020년 등) 자료를 최신 기준(2025년 등) 파일로 자동 교체.

## 4. [문서 관리] 견적서 파일 자동 정리 및 이메일 요청 연동

**상세 내용:** 하루 20~30개씩 발생하는 견적서 파일을 공통 폴더 및 스프레드시트에 자동 정리하고, 이메일로 오는 견적 요청을 시스템에 즉시 등록해야 합니다.

**체크리스트:**
- [ ] 견적서 파일을 공통 폴더 내 정해진 규칙에 따라 자동 분류 및 저장.
- [ ] 이메일로 접수된 견적 의뢰 내용을 감지하여 시스템 현황판에 자동 등록.
- [ ] 과거 유사 실적 정보를 키워드 기반으로 AI가 자동 추출하여 견적서 작성에 활용.

## 5. [협업] 일일 회의 체계 및 회의록 자동 정리

**상세 내용:** 원장님과의 의사결정 간극을 줄이기 위해 매일 1회 정기 회의를 진행하고, 모든 논의 내용을 즉시 문서화해야 합니다.

**체크리스트:**
- [ ] 매일 오전/오후 정기 회의를 통한 의견 전달 및 진행 상황 파악 루틴 확립.
- [ ] 회의 녹음 데이터를 즉시 텍스트로 변환(STT)하고 파일로 정리하여 히스토리 관리.
- [ ] 주요 업무 판단 기준(20~30가지)을 디지털 매뉴얼화하여 노트북LM으로 검증.
