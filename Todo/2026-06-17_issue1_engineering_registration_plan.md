**체크리스트:**
- [ ] 등록 분야(사업관리) 필수인력 요건 확정: 인원수(10명) 및 분야별 등급 분포 요건.
- [x] 기술등급 판정표(자격 기준 / 학력 기준 / 경력 기준) 최신본 확보 전 단계의 내부 판정 기준 초안 정리.

**체크리스트:**

- [ ] 학력(전공)과 등록 분야 적합성 매핑 기준 정리.

**상세 내용:** 자격·학력·경력 조합으로 초급/중급/고급/특급을 산출하는 로직을 정의해 quali-fit 점수화 체계에 연결한다.

## 6. [2026-06-18 실행 결과] 1차 실행 검토

**실행 내용:** `quali-fit/app.db` 기준 인력·학력·자격 데이터를 비식별로 집계하고, 엔지니어링 등록 검토용 후보군/보강 입력 양식을 생성했다.

**결론:** 현재 DB에는 경력연수 데이터가 없어 최소 10명 요건 충족 여부와 인별 기술등급은 확정할 수 없다. 다음 단계는 `Input_Template` 시트에 엔지니어링 경력연수, 사업관리/PM 경력연수, 증빙 보유 여부를 채운 뒤 2차 판정을 수행하는 것이다.

**비식별 후보군 집계(app.db 기준):**
- 기술사급 우선 후보: 4명
- 기사/산업기사 기준 후보: 12명
- 학사 이상 학력 기준 후보: 19명
- 데이터 보강 필요: 6명

**진전 가능성:** 최소 10명 후보를 찾는 1차 선별 단계는 `app.db`만으로 진행 가능하다. 다만 최종 등록 가능 여부는 경력연수와 증빙 여부가 입력된 뒤 확정한다.

**다음 입력 항목:**
- 엔지니어링 관련 경력연수
- 사업관리/PM 관련 경력연수
- 자격증·학력·경력 증빙서류 보유 여부
- 등록 분야 적합성 확인
- 기존 협회 등록 여부
