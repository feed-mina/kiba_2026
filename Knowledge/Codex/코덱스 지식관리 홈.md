# 코덱스 지식관리 홈

KIBA에서 Codex가 처리한 질문, 결정, 후속 작업을 Obsidian에서 탐색하기 위한 시작점입니다.

## 바로가기

- [[Knowledge/이슈 중심 프로젝트 지식관리|이슈 중심 프로젝트 지식관리]]
- [[Codex Index]]: ASK/Todo에서 뽑은 Codex 관련 항목 자동 인덱스
- [[Codex MOC]]: 전체 지도
- [[Codex Topics - Automation]]: 스케줄러, 자동 기록, 반복 작업
- [[Codex Topics - Docs and R2]]: docs/R2 미러링, Worker, DPAPI
- [[Codex Topics - GitHub Ops]]: GitHub Issue, Projects, Pages, git 운영

## 운영 방식

- 원본 기록은 계속 `ASK/`와 `Todo/`에 남깁니다.
- Obsidian에서는 이 폴더의 인덱스와 주제 노트로 찾고 묶습니다.
- 세부 사실은 원본 ASK/Todo 링크를 타고 확인합니다.
- 오래 유지할 판단, 원칙, 재발 방지 메모는 주제 노트에 짧게 남깁니다.

## 갱신

```powershell
python .\scripts\build_codex_obsidian.py
```

일일 스케줄러가 실행될 때도 Codex 인덱스가 자동으로 다시 만들어지도록 연결할 수 있습니다.
