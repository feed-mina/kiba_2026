## 운영 원칙

- GitHub Issue: 실행과 상태의 기준
- `index.html`: 한눈에 보는 현재 상태
- Obsidian: 결정, 맥락, 날짜별 흐름, docs 링크
- `docs/`: 원문 보관. Obsidian에는 원문 복사 없이 메타데이터와 링크만 유지
