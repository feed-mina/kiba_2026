# ASK 로그 관리

Claude와 Codex에게 한 질문과 답변을 하루 단위로 요약해 보관하는 폴더입니다.

## 파일 이름

```text
YYYY-MM-DD_ai.md
```

예:

```text
2026-06-17_ai.md
```

## 기록 대상

- Claude와 나눈 질문·응답
- Codex와 나눈 질문·응답
- 두 도구가 같은 주제에서 이어서 작업한 내용

## 작성 규칙

질문마다 한 블록으로 기록합니다.

```markdown
## 주제 요약
**도구:** Claude 또는 Codex
**질문:** 한 줄 요약
**응답 요약:**
- 핵심 응답 1
- 핵심 응답 2
- 필요하면 다음 할 일
```

## 매일 쓰는 요청 문장

```text
오늘 Claude/Codex와 나눈 대화를 ASK 형식으로 요약해서 오늘자 ai 로그에 누적해줘.
```

특정 도구만 정리하고 싶을 때:

```text
오늘 Codex와 나눈 대화만 ASK 형식으로 요약해줘.
오늘 Claude와 나눈 대화만 ASK 형식으로 요약해줘.
```
