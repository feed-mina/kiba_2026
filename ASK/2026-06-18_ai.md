## 견적서 생성기 exe 빌드 단계별 상세 안내
**도구:** Claude
**질문:** `pip install` → PyInstaller 빌드가 자동 진행되는 부분부터 어떻게 하는지 자세하게 알려달라.
**응답 요약:**
- `build.bat` 더블클릭이 [1] Python 확인 → [2] `pip install`(openpyxl·pyinstaller) → [3] 이전 산출물 제거 → [4] PyInstaller `--onefile --noconsole` 빌드의 4단계를 자동 수행하며, 각 단계 화면 로그와 성공 메시지(`Successfully installed`, `completed successfully`)를 안내.
- 사내망 설치 실패 시 `--proxy` / `--trusted-host` 옵션, 결과물은 `dist\견적서생성기.exe`(15~25MB) 한 파일만 배포하면 받는 PC에 Python 불필요.
- 흔한 문제 대처: SmartScreen 경고("추가 정보→실행"), 백신 오탐(`--noupx`), 무반응 시 `--noconsole` 제거로 에러 확인, 한글 깨짐 `chcp 65001`.
